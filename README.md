# nurture-design-system

Design system reference for Nurture, the product design and research lab at Arbisoft.

**What's in here**

**DESIGN.md** — the full Nurture design system: colors, typography, spacing, components, and dos/don'ts. Written in a format AI coding agents can read and apply directly.

**CLAUDE.md** — instructs Claude Code to read DESIGN.md before generating any UI.


**How to use it**

**With Claude Code**

Copy DESIGN.md and CLAUDE.md into the root of your project. Claude Code will read them automatically on launch and apply the Nurture design system to any UI it generates.

**With Cursor or Windsurf**

Copy DESIGN.md into your project root. Reference it in your .cursorrules file or equivalent config.

**With Claude.ai**

Add DESIGN.md as a file in your Claude Project. It will stay in context across all conversations in that project.

**Reference without copying**

**Add this to your project's CLAUDE.md to pull the latest version directly:**

Always apply the Nurture design system defined at:

https://raw.githubusercontent.com/ibrahimkhan95/nurture-design-system/main/DESIGN.md

**Updating the system**
Open a pull request with your changes. One reviewer, then merge. Changes are live for everyone immediately.
